# NewCool Pesca - SERNAPESCA Chile

Portal educativo sobre pesca y SERNAPESCA en Chile.

## Demo

[https://pesca.newcool.io](https://pesca.newcool.io)

## Caracteristicas

- Buscador de oficinas SERNAPESCA
- Guia de permisos de pesca
- Calendario de vedas
- Catalogo de especies
- Calculadora de licencias

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
